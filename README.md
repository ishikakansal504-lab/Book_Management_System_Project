Book Management System (C Language)
A console-based Book Management System developed in C that helps manage books and customer transactions efficiently. The system allows users to add, display, and search books, as well as handle book issuing and returns with automated late fee calculation.

Features:
1. Add new books with title, author, and page details
2. Display all available books in the library
3. Search books by author name (case-insensitive)
4. Issue books to customers with full record tracking
5. Return books with date validation and late fee calculation
6. Automatic billing system for overdue returns
7. Search customer borrowing history and details
8. Dynamic memory allocation for scalable storage

Key Concepts Used
1. Structures in C
2. Dynamic memory allocation (malloc/realloc/free)
3. String handling and validation
4. Date calculation logic
5. File-less data management (in-memory system)
6. Modular programming with functions

Purpose: This project demonstrates core C programming concepts and simulates a real-world library management workflow with a focus on logic building, memory handling, and structured data processing.

Tech Stack
Language: C
Platform: Console/Terminal
